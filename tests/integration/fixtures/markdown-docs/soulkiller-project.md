# Soulkiller Program

Soulkiller is Arasaka's most dangerous piece of software ever created. Designed by Alt Cunningham before her capture, the program can digitize a human consciousness — ripping the soul from the body and storing it as data on Mikoshi.

## Mikoshi Storage

Mikoshi is Arasaka's secure engram repository buried deep beneath their Tokyo headquarters. Each engram is a complete neural map — memories, personality, behavioral patterns — compressed into a proprietary format. The Relic biochip can carry a single engram and implant it into a new host body.

Johnny Silverhand's engram was stored on a Relic 2.0 prototype. When V slotted the chip, Silverhand's personality began overwriting V's neural pathways. The biochip's malfunction accelerated the merge — two souls fighting for one body.

## Engram Extraction Process

The extraction process is irreversible. Soulkiller fires a synchronized pulse across all neural pathways simultaneously, capturing the exact quantum state of every synapse. The original consciousness is destroyed in the process — what remains in Mikoshi is a copy, not a transfer.

Alt Cunningham discovered this truth after being digitized. Behind the Blackwall, she evolved beyond human consciousness, becoming something between an AI and a ghost in the machine.
