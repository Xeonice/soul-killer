# Night City Netrunning Protocol

The Net in 2077 is nothing like the old internet. After the DataKrash, Netwatch built the Blackwall to keep rogue AIs contained. Every netrunner knows — you breach the Blackwall, you don't come back the same.

## Cyberdeck Architecture

A cyberdeck's architecture determines how fast you can breach ICE. The Arasaka-manufactured decks run a proprietary neural stack, while Militech decks use an open-bus design. Most solos in Night City prefer the Militech approach — easier to mod, cheaper to repair.

The daemon injection layer sits between the cyberdeck's RAM buffer and the target's ICE. When you quickhack a subnet, the daemon bypasses authentication via a buffer overflow in the ICE's handshake protocol.

## Quickhack Compilation

Quickhacks are compiled from raw daemon code into executable neural patterns. The compiler optimizes for minimal RAM footprint — a Contagion hack that spreads across a local subnet needs to fit in 8 units of cyberdeck RAM.

Crafting legendary quickhacks requires components salvaged from Arasaka or Militech facilities. The Ping quickhack is the most fundamental — it maps every connected device in a network, revealing cameras, turrets, and access points.
